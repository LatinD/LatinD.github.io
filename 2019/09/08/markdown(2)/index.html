    • 七、列表

    ·无序列表

    语法：
    无序列表用 - + * 任何一种都可以
    - 列表内容
    + 列表内容
    * 列表内容

    注意：- + * 跟内容之间都要有一个空格

    • 列表内容

      • 列表内容
    • 列表内容
      无序列表标记符前每增加两个空格，增加一个层级：

      ·有序列表

      则使用数字接着一个英文句点。标记符号后面也要接着至少一个空格或制表符(Tab)。在列表标记上使用的数字并不会影响输出的 HTML 结果。
      示例：

    \1. First paragraph
    \2. Second paragraph
    \5. Third paragraph

    效果：

    1. First paragraph
    2. Second paragraph
    3. Third paragraph

    有序列表标记符前每增加一个 Tab，增加一个层级：

    八、任务列表

    代码：
    - [ ] a task list item
    - [x] completed
    - [ ] incomplete
    - [ ] list syntax required
    - [x] completed

    ps:代码2 3 行前面要空一格表示下一级
    显示效果：

    • a task list item
      • completed
      • incomplete
    • list syntax required
    • completed

    九、表格

    |表头|表头|表头|
|---|---|---|
|内容|内容|内容|
|内容|内容|内容|

第二行分割表头和内容。
- 有一个就行，为了对齐，多加了几个
文字默认居左
-两边加：表示文字居中
-右边加：表示文字居右
注：原生的语法两边都要用 | 包起来。此处省略

    十、代码

    语法：
    单行代码：代码之间分别用一个反引号(键盘左上角)包起来
    daima
    daima
    代码块：代码之间分别用三个反引号包起来，且两边的反引号单独占一行
